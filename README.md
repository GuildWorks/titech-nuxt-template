### node install(For Mac)
1. install home brew

```$xslt
$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

2. install node
```$xslt
$ brew install nodebrew
$ nodebrew install-binary latest
```

## yarn install
1. yarn install
```$xslt
$ npm install yarn -g
```
2. yarnの確認コマンド（インストールされたyarnのバージョンが表示される）
```$xslt
$ yarn --version
```

## Initial launch
1. Install libraries
```$xslt
$ yarn install
```


## Launch application
```
$ yarn serve
```
